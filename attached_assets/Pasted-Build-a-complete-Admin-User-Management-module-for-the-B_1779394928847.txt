Use the existing API endpoints below.

---

# API ENDPOINTS

## 1. GET ALL USERS

Endpoint:

```txt
GET /api/Profiles
```

This endpoint returns all profiles.

Use this endpoint to populate the admin users table.

---

## 2. GET USER PROFILE DETAILS

Endpoint:

```txt
GET /api/Profiles/{id}
```

Where:

* `{id}` = ProfileId

Use this to load internal detailed profile information.

---

## 3. GET USER PUBLIC DOSSIER

Endpoint:

```txt
GET /api/Profiles/user-public-profile?userId=
```

Use:

* userId from the selected profile

This endpoint returns:

* professional title
* verified experience
* badges
* completed tasks
* leaderboard rank
* XP
* public professional profile

Use this for the Public Dossier/Profile tab.

---

## 4. RESET USER PASSWORD

Endpoint:

```txt
GET /api/Account/reset_user_password?email={email}&userId={adminUserId}
```

IMPORTANT:

* Pass the selected user's email
* Pass the currently logged in admin userId

After successful reset:

* Show success notification
* Inform admin that a temporary password email has been sent
