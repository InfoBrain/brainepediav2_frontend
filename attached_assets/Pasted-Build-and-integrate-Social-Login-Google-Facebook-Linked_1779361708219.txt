Build and integrate Social Login (Google, Facebook, LinkedIn, GitHub if available) into the Brainepedia Login and Sign Up pages.

OBJECTIVE

Allow users to authenticate using social providers through browser redirection using the backend endpoint:

GET /api/Account/social-login/{provider}

Example:

window.location.href =
"https://api.brainepedia.com/api/Account/social-login/Google?redirectUrl=https://demo.brainepedia.com/login-success";

Do NOT use fetch or axios for social login initialization.
It must use full browser redirect navigation.

3. CREATE SOCIAL LOGIN HELPER

Create reusable helper:

const API_BASE = "https://api.brainepedia.com";

const socialLogin = (provider) => {
  const redirectUrl =
    encodeURIComponent("https://demo.brainepedia.com/login-success");

  window.location.href =
    `${API_BASE}/api/Account/social-login/${provider}?redirectUrl=${redirectUrl}`;
};
