Design a focused, distraction-free problem-solving workspace where users can:

Read the mission while working
Write their solution (code + explanation)
Attach evidence files
Ask AI for help (Brainiac)
Submit their work for evaluation
📄 PAGE ROUTE
/app/session/:sessionId/solve
🔗 API INTEGRATION
🔷 1. GET SESSION
GET /api/ExperienceSessions/{id}
🔷 2. SUBMIT WORK
POST /api/Submissions/submit

FormData:

ExperienceSessionId
ApproachExplanation
CodeSnippet
EvidenceFiles[]
🔷 3. ASK AI (BRAINIAC)
POST /api/Evaluations/ask-brainiac

Body:

{
  "sessionId": "guid",
  "userId": "string",
  "currentApproach": "text",
  "currentCode": "text"
}
