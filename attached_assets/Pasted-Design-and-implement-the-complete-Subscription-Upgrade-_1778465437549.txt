🔷 API ENDPOINTS
Initialize Upgrade

POST

/api/Subscriptions/initialize-upgrade

Body:

{
  "userId": "string",
  "newTier": "Architect"
}

OR

{
  "userId": "string",
  "newTier": "Grandmaster"
}

Response:

{
  "checkoutUrl": "https://checkout.paystack.com/..."
}
Verify Payment

GET

/api/Subscriptions/verify-payment?reference=xxxx

Response:

"Subscription Activated!"
