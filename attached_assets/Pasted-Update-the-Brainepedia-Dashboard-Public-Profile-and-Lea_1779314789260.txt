Update the Brainepedia Dashboard, Public Profile, and Leaderboard system to fully integrate the new Verified Experience (VX) architecture and professional identity system.

OBJECTIVE
Implement a modern professional identity experience powered by:

* Verified Experience Years (VX)
* Professional Rank Titles
* Public Shareable Dossier Profiles
* Recruiter-Friendly Public Profile Pages
* Clickable Leaderboard Profiles

BACKEND ENDPOINTS TO USE

1. VERIFIED EXPERIENCE ENDPOINT
   GET /api/Profile/professional-identity?userId={userId}

Response:
{
"fullName": "",
"avatarUrl": "",
"activeProfession": "",
"totalAccumulatedXp": 0,
"verifiedExperienceYears": 0,
"professionalTitle": "",
"displayString": ""
}

2. PUBLIC PROFILE ENDPOINT
   GET /api/Profile/user-public-profile?userId={userId}

Response contains:

* DisplayName
* ProfilePictureUrl
* ActiveProfession
* ProfessionalTitle
* TotalXp
* VerifiedExperienceYears
* GlobalLeaderboardRank
* EarnedBadges[]
* CompletedMissions[]
